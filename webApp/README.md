# Biashara360 Web App

React 18 + TypeScript + Vite + Tailwind CSS

## Quick Start
```bash
cd webApp/
npm install
npm run dev
```
Open http://localhost:3000

## Production Build
```bash
npm run build   # outputs to dist/
```

## Environment
Edit `.env` to point at your backend:
```
VITE_API_BASE_URL=http://localhost:8080/v1
```
